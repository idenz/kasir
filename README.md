## Requirement
- Nodejs v14^
- MongoDB

### Setup project
```sh
# clone project
> git clone https://github.com/idenz/kasir.git

# Postman collection
> https://www.getpostman.com/collections/659c0a9fb479b1d19100
```

### Start Project

```sh
# Install package
> npm install

# API running
> npm start
```

### Step by step POSTMAN Collection

```sh
# Register user
> Auth
    > Register

# Login user
> Auth
    > Login
```
